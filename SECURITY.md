# Security Policy

## Supported versions

This project is pre-1.0. Please use the latest released version.

## Reporting a vulnerability

Please do **not** open a public GitHub issue for security vulnerabilities.

Instead, contact the maintainer privately via GitHub profile:

- https://github.com/Cordedmink2

Include:

- affected plugin version
- clear reproduction steps
- impact summary
- any proof-of-concept details

We will acknowledge reports as quickly as possible and coordinate a fix + release.
